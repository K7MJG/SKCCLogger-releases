# SKCCLogger v2 - Downloads

Log manager for the [Straight Key Century Club (SKCC)](https://skccgroup.com).

## Downloads

**[Go to Releases](https://github.com/K7MJG/SKCCLogger-releases/releases/latest)** to download the latest version for your platform.

## Supported Platforms

| Platform | File |
|---|---|
| Windows (64-bit) | `skcclogger-windows-amd64.exe` |
| Windows (32-bit) | `skcclogger-windows-x86.exe` |
| macOS (Apple Silicon) | `skcclogger-macos-arm64.tar.gz` |
| macOS (Intel) | `skcclogger-macos-intel.tar.gz` |
| Linux (64-bit) | `skcclogger-linux-amd64` or `.deb` / `.rpm` |
| Linux (ARM 64-bit) | `skcclogger-linux-arm64` or `.deb` / `.rpm` |
| Linux (32-bit) | `skcclogger-linux-x86` |
| Linux (Raspberry Pi) | `skcclogger-linux-armhf` |

## Installation

**Windows:** Download the `.exe` and run it. No installer needed.

**macOS:** Download the `.tar.gz`, extract, and move `SKCCLogger v2.app` to your Applications folder. On first launch, right-click and choose Open to bypass Gatekeeper.

**Linux:** Download the binary and make it executable (`chmod +x skcclogger-linux-*`), or install the `.deb` / `.rpm` package.
